🍎 Deploy on MacBook

  1. Save this file anywhere — e.g. ~/Desktop/bol-zombai.html
  2. Open Terminal (Cmd+Space → "Terminal")
  3. Navigate to the folder: cd ~/Desktop
  4. Start a local server: python3 -m http.server 8080
  5. Open browser → http://localhost:8080/bol-zombai.html
  6. Allow microphone permissions when prompted 🎤
  7. Paste your API keys in the config accordion — they stay in memory only

🔑 Getting Your API Keys

  • Anthropic Key: console.anthropic.com → API Keys → Create Key. Starts with sk-ant-
  • ElevenLabs Key: elevenlabs.io → Profile → API Key section
  • Voice ID: ElevenLabs → Voices → click voice → copy ID from URL. Default is "Rachel"
  • Keys are entered in the UI and only used in-browser — never stored on disk

⚡ Quick Notes

  • Zero dependencies — single HTML file, works anywhere
  • Best on Chrome or Edge (strongest mic / Speech API support)
  • Firefox works for chat; mic features need a local server due to browser security

📜 How To Run Bol-ZombAI

  1. Click the accordion header to expand API Keys & Persona config
  2. Enter your Anthropic API Key
  3. Enter your ElevenLabs API Key
  4. Optionally change the Voice ID to another ElevenLabs voice
  5. Choose a Persona — Bol-ZombAI speaks as an undead oracle by default
  6. Click the red 🎤 SPEAK button — allow mic permission when asked
  7. Speak your question clearly into the mic
  8. Click 🛑 STOP when done speaking
  9. Bol-ZombAI transcribes → sends to Claude → speaks back via ElevenLabs 🔊
  10. The audio player appears and auto-plays the undead response!
